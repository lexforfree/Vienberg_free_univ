## ЧАСТЬ 1: ГРУППЫ

### Задача 1.1: Проверка группы (ℤ, +)

**Условие:**
Проверьте, что множество целых чисел ℤ с операцией сложения образует группу. Укажите нейтральный элемент и обратный элемент для элемента 5.

**Разбор:**
Нужно проверить четыре аксиомы группы:
1. **Замкнутость:** Если a, b ∈ ℤ, то a + b ∈ ℤ ✓
2. **Ассоциативность:** (a + b) + c = a + (b + c) для всех a, b, c ∈ ℤ ✓
3. **Нейтральный элемент:** e = 0, так как a + 0 = a для всех a ∈ ℤ ✓
4. **Обратные элементы:** Для каждого a ∈ ℤ существует -a ∈ ℤ такой, что a + (-a) = 0 ✓

Нейтральный элемент: **0**
Обратный элемент для 5: **-5** (так как 5 + (-5) = 0)

### Задача 1.2: Проверка группы (ℝ\{0}, ·)

**Условие:**
Является ли множество ненулевых действительных чисел ℝ\{0} с операцией умножения группой? Укажите нейтральный элемент и обратный элемент для элемента 3.

**Разбор:**
1. **Замкнутость:** Если a, b ∈ ℝ\{0}, то a · b ∈ ℝ\{0} (произведение ненулевых чисел ненулевое) ✓
2. **Ассоциативность:** (a · b) · c = a · (b · c) для всех a, b, c ∈ ℝ\{0} ✓
3. **Нейтральный элемент:** e = 1, так как a · 1 = a для всех a ∈ ℝ\{0} ✓
4. **Обратные элементы:** Для каждого a ∈ ℝ\{0} существует 1/a ∈ ℝ\{0} такой, что a · (1/a) = 1 ✓

**Ответ:** Да, это группа.
Нейтральный элемент: **1**
Обратный элемент для 3: **1/3** (так как 3 · (1/3) = 1)

### Задача 1.3: Проверка подгруппы

**Условие:**
Является ли множество четных целых чисел 2ℤ = {..., -4, -2, 0, 2, 4, ...} подгруппой группы (ℤ, +)?

**Разбор:**
Для проверки подгруппы используем критерий: H ⊆ G — подгруппа ⟺ для всех a, b ∈ H выполнено a · b⁻¹ ∈ H.

1. **2ℤ ⊆ ℤ:** Очевидно ✓
2. **Для a, b ∈ 2ℤ:** a = 2m, b = 2n для некоторых m, n ∈ ℤ
3. **a - b = 2m - 2n = 2(m - n) ∈ 2ℤ** ✓

**Ответ:** Да, 2ℤ — подгруппа (ℤ, +).

### Задача 1.4: Порядок элемента в конечной группе

**Условие:**
Найдите порядок элемента 3 в группе (ℤ₁₂, +) (остатки от деления на 12 со сложением по модулю 12).

**Разбор:**
Порядок элемента g — это наименьшее натуральное число n такое, что gⁿ = e (в аддитивной группе: ng = 0).

Для элемента 3 в ℤ₁₂:
- 1 · 3 = 3 ≠ 0
- 2 · 3 = 6 ≠ 0
- 3 · 3 = 9 ≠ 0
- 4 · 3 = 12 ≡ 0 (mod 12) ✓

**Ответ:** ord(3) = **4**

### Задача 1.5: Антипример группы

**Условие:**
Является ли множество натуральных чисел ℕ = {1, 2, 3, ...} с операцией сложения группой?

**Разбор:**
Проверим аксиомы:
1. **Замкнутость:** Если a, b ∈ ℕ, то a + b ∈ ℕ ✓
2. **Ассоциативность:** (a + b) + c = a + (b + c) ✓
3. **Нейтральный элемент:** Нужен элемент e такой, что a + e = a для всех a ∈ ℕ. Это должно быть 0, но 0 ∉ ℕ ✗

**Ответ:** Нет, не является группой (нет нейтрального элемента).

### Задача 1.6: Абелева группа

**Условие:**
Проверьте, что группа (ℚ\{0}, ·) абелева (коммутативна).

**Разбор:**
Группа G называется абелевой, если ab = ba для всех a, b ∈ G.

Для ненулевых рациональных чисел: если a = p₁/q₁, b = p₂/q₂, то
a · b = (p₁/q₁) · (p₂/q₂) = (p₁p₂)/(q₁q₂) = (p₂p₁)/(q₂q₁) = (p₂/q₂) · (p₁/q₁) = b · a

(Используем коммутативность умножения целых чисел)

**Ответ:** Да, группа (ℚ\{0}, ·) абелева.

### Задача 1.7: Циклическая подгруппа

**Условие:**
Найдите циклическую подгруппу, порожденную элементом 2 в группе (ℤ₁₀, +).

**Разбор:**
Циклическая подгруппа ⟨2⟩ состоит из всех степеней элемента 2 (в аддитивной группе — всех кратных 2):

- 0 · 2 = 0
- 1 · 2 = 2
- 2 · 2 = 4
- 3 · 2 = 6
- 4 · 2 = 8
- 5 · 2 = 10 ≡ 0 (mod 10)

Цикл повторяется.

**Ответ:** ⟨2⟩ = {0, 2, 4, 6, 8}, |⟨2⟩| = 5

## ЧАСТЬ 2: ЦИКЛИЧЕСКИЕ ГРУППЫ

### Задача 2.1: Циклическая группа ℤₙ

**Условие:**
Докажите, что группа (ℤ₅, +) циклична и найдите все порождающие элементы.

**Разбор:**
ℤ₅ = {0, 1, 2, 3, 4} со сложением по модулю 5.

Проверим, что 1 — порождающий элемент:
- 1 · 1 = 1
- 2 · 1 = 2
- 3 · 1 = 3
- 4 · 1 = 4
- 5 · 1 = 0

Итак, ⟨1⟩ = ℤ₅, поэтому группа циклична.

Порождающие элементы — это элементы порядка 5. Используем формулу: ord(k) = n/gcd(k, n).

- ord(1) = 5/gcd(1,5) = 5/1 = 5 ✓
- ord(2) = 5/gcd(2,5) = 5/1 = 5 ✓
- ord(3) = 5/gcd(3,5) = 5/1 = 5 ✓
- ord(4) = 5/gcd(4,5) = 5/1 = 5 ✓

**Ответ:** Все элементы {1, 2, 3, 4} — порождающие. Количество: φ(5) = 4.

---

### Задача 2.2: Порождающие элементы в ℤₙ

**Условие:**
Найдите все порождающие элементы группы (ℤ₁₂, +).

**Разбор:**
Порождающие элементы в ℤₙ — это элементы, взаимно простые с n.

Для ℤ₁₂: нужны элементы k такие, что gcd(k, 12) = 1.

Проверим:
- gcd(1, 12) = 1 ✓
- gcd(5, 12) = 1 ✓
- gcd(7, 12) = 1 ✓
- gcd(11, 12) = 1 ✓

Остальные элементы не взаимно просты с 12.

**Ответ:** Порождающие элементы: {1, 5, 7, 11}. Количество: φ(12) = 4.

### Задача 2.3: Подгруппы циклической группы

**Условие:**
Найдите все подгруппы группы (ℤ₁₂, +).

**Разбор:**
В циклической группе порядка n каждая подгруппа имеет порядок, делящий n, и есть ровно одна подгруппа каждого делящего n порядка.

Делители 12: 1, 2, 3, 4, 6, 12.

Подгруппы:
- Порядок 1: {0}
- Порядок 2: ⟨6⟩ = {0, 6}
- Порядок 3: ⟨4⟩ = {0, 4, 8}
- Порядок 4: ⟨3⟩ = {0, 3, 6, 9}
- Порядок 6: ⟨2⟩ = {0, 2, 4, 6, 8, 10}
- Порядок 12: ⟨1⟩ = ℤ₁₂

**Ответ:** 6 подгрупп: {0}, ⟨6⟩, ⟨4⟩, ⟨3⟩, ⟨2⟩, ℤ₁₂.

### Задача 2.4: Корни из единицы

**Условие:**
Докажите, что множество корней n-й степени из единицы μₙ = {z ∈ ℂ : zⁿ = 1} образует циклическую группу порядка n относительно умножения.

**Разбор:**
Элементы μₙ: ζₖ = e^(2πik/n) для k = 0, 1, ..., n-1.

1. **Замкнутость:** ζⱼ · ζₖ = e^(2πij/n) · e^(2πik/n) = e^(2πi(j+k)/n) = ζ_{j+k mod n} ∈ μₙ ✓

2. **Цикличность:** ζ₁ = e^(2πi/n) — порождающий элемент:
   - ζ₁⁰ = 1
   - ζ₁¹ = e^(2πi/n)
   - ζ₁² = e^(4πi/n)
   - ...
   - ζ₁ⁿ = e^(2πi) = 1

Итак, ⟨ζ₁⟩ = μₙ.

**Ответ:** μₙ ≅ ℤₙ — циклическая группа порядка n.

### Задача 2.5: Изоморфизм циклических групп

**Условие:**
Докажите, что любые две циклические группы одинакового порядка изоморфны.

**Разбор:**
Пусть G = ⟨g⟩ и H = ⟨h⟩ — циклические группы порядка n.

Определим отображение φ: G → H по формуле φ(gᵏ) = hᵏ.

1. **Корректность:** Если gᵃ = gᵇ, то a ≡ b (mod n), поэтому hᵃ = hᵇ ✓
2. **Гомоморфизм:** φ(gᵃ · gᵇ) = φ(gᵃ⁺ᵇ) = hᵃ⁺ᵇ = hᵃ · hᵇ = φ(gᵃ) · φ(gᵇ) ✓
3. **Биективность:** Каждый элемент H имеет вид hᵏ = φ(gᵏ), и φ инъективна (из определения) ✓

**Ответ:** φ — изоморфизм, поэтому G ≅ H.

### Задача 2.6: Порядок элемента в циклической группе

**Условие:**
В циклической группе ℤ₂₀ найдите порядок элемента 8.

**Разбор:**
Используем формулу: ord(k) = n/gcd(k, n).

Для k = 8, n = 20:
gcd(8, 20) = gcd(8, 20) = 4

ord(8) = 20/4 = 5

Проверка:
- 1 · 8 = 8
- 2 · 8 = 16
- 3 · 8 = 24 ≡ 4 (mod 20)
- 4 · 8 = 32 ≡ 12 (mod 20)
- 5 · 8 = 40 ≡ 0 (mod 20) ✓

**Ответ:** ord(8) = **5**

## ЧАСТЬ 3: СМЕЖНЫЕ КЛАССЫ И ТЕОРЕМА ЛАГРАНЖА

### Задача 3.1: Смежные классы в ℤ

**Условие:**
Найдите все левые смежные классы группы (ℤ, +) по подгруппе 3ℤ = {..., -6, -3, 0, 3, 6, ...}.

**Разбор:**
Левый смежный класс элемента a по подгруппе H: a + H = {a + h : h ∈ H}.

Для a ∈ ℤ и H = 3ℤ:
- 0 + 3ℤ = {..., -6, -3, 0, 3, 6, ...} (остаток 0)
- 1 + 3ℤ = {..., -5, -2, 1, 4, 7, ...} (остаток 1)
- 2 + 3ℤ = {..., -4, -1, 2, 5, 8, ...} (остаток 2)
- 3 + 3ℤ = 0 + 3ℤ (повторение)

Смежные классы определяются остатком от деления на 3.

**Ответ:** Три смежных класса:
- 0 + 3ℤ (остаток 0)
- 1 + 3ℤ (остаток 1)
- 2 + 3ℤ (остаток 2)

---

### Задача 3.2: Смежные классы в ℤₙ

**Условие:**
Найдите все смежные классы группы (ℤ₁₂, +) по подгруппе H = ⟨3⟩ = {0, 3, 6, 9}.

**Разбор:**
Подгруппа H = ⟨3⟩ = {0, 3, 6, 9}, |H| = 4.

Смежные классы:
- 0 + H = {0, 3, 6, 9}
- 1 + H = {1, 4, 7, 10}
- 2 + H = {2, 5, 8, 11}
- 3 + H = {3, 6, 9, 0} = 0 + H (повторение)

Количество смежных классов: [ℤ₁₂ : H] = |ℤ₁₂|/|H| = 12/4 = 3.

**Ответ:** Три смежных класса:
- {0, 3, 6, 9}
- {1, 4, 7, 10}
- {2, 5, 8, 11}

---

### Задача 3.3: Смежные классы в S₃

**Условие:**
Найдите все левые смежные классы группы S₃ (перестановки трех элементов) по подгруппе A₃ = {e, (123), (132)} (четные перестановки).

**Разбор:**
S₃ имеет 6 элементов: e, (12), (13), (23), (123), (132).
A₃ имеет 3 элемента: e, (123), (132).

Левые смежные классы:
- e · A₃ = {e, (123), (132)} (четные перестановки)
- (12) · A₃ = {(12), (12)(123), (12)(132)} = {(12), (23), (13)} (нечетные)
- (13) · A₃ = {(13), (13)(123), (13)(132)} = {(13), (12), (23)} = (12) · A₃ (повторение)

Количество смежных классов: [S₃ : A₃] = 6/3 = 2.

**Ответ:** Два смежных класса:
- A₃ = {e, (123), (132)} (четные перестановки)
- (12)A₃ = {(12), (23), (13)} (нечетные перестановки)

### Задача 3.4: Применение теоремы Лагранжа

**Условие:**
Используя теорему Лагранжа, найдите все возможные порядки элементов в группе S₃ (порядок 6).

**Разбор:**
По теореме Лагранжа, порядок любого элемента делит порядок группы.

Делители 6: 1, 2, 3, 6.

Проверим элементы S₃:
- ord(e) = 1 (нейтральный элемент)
- ord((12)) = 2, ord((13)) = 2, ord((23)) = 2 (транспозиции)
- ord((123)) = 3, ord((132)) = 3 (3-циклы)
- Нет элемента порядка 6

**Ответ:** Возможные порядки: 1, 2, 3.

### Задача 3.5: Индекс подгруппы

**Условие:**
Найдите индекс подгруппы H = ⟨(123)⟩ в группе S₃.

**Разбор:**
H = ⟨(123)⟩ = {e, (123), (132)}, |H| = 3.
|S₃| = 6.

По теореме Лагранжа:
[S₃ : H] = |S₃|/|H| = 6/3 = 2

Это означает, что S₃ разбивается на 2 смежных класса по H.

**Ответ:** [S₃ : H] = **2**

### Задача 3.6: Следствие теоремы Лагранжа

**Условие:**
Докажите, что в группе простого порядка p каждый неединичный элемент имеет порядок p.

**Разбор:**
Пусть G — группа порядка p (p простое), g ∈ G, g ≠ e.

По теореме Лагранжа, ord(g) делит |G| = p.

Так как p простое, его делители: 1 и p.

Так как g ≠ e, то ord(g) ≠ 1.

Следовательно, ord(g) = p.

**Вывод:** Каждый неединичный элемент порождает всю группу G, поэтому G циклична.

**Ответ:** ord(g) = p для всех g ≠ e.

### Задача 3.7: Теорема Лагранжа и циклические группы

**Условие:**
Найдите все подгруппы циклической группы ℤ₃₀.

**Разбор:**
По теореме Лагранжа, порядок подгруппы делит порядок группы.

Делители 30: 1, 2, 3, 5, 6, 10, 15, 30.

В циклической группе есть ровно одна подгруппа каждого порядка, делящего порядок группы:

- Порядок 1: {0}
- Порядок 2: ⟨15⟩ = {0, 15}
- Порядок 3: ⟨10⟩ = {0, 10, 20}
- Порядок 5: ⟨6⟩ = {0, 6, 12, 18, 24}
- Порядок 6: ⟨5⟩ = {0, 5, 10, 15, 20, 25}
- Порядок 10: ⟨3⟩ = {0, 3, 6, 9, 12, 15, 18, 21, 24, 27}
- Порядок 15: ⟨2⟩ = {0, 2, 4, 6, ..., 28}
- Порядок 30: ℤ₃₀

**Ответ:** 8 подгрупп, соответствующих делителям 30.

### Задача 3.8: Нормальные подгруппы

**Условие:**
Является ли подгруппа A₃ нормальной в S₃?

**Разбор:**
Подгруппа H нормальна в G, если gH = Hg для всех g ∈ G.

Для S₃ и A₃:
- Индекс [S₃ : A₃] = 2

Ключевое свойство: подгруппа индекса 2 всегда нормальна.

Доказательство: если [G : H] = 2, то есть ровно два смежных класса: H и gH (для любого g ∉ H). Тогда G = H ∪ gH = H ∪ Hg, откуда gH = Hg.

**Ответ:** Да, A₃ нормальна в S₃ (так как [S₃ : A₃] = 2).

## ИТОГОВЫЕ ФОРМУЛЫ И КЛЮЧЕВЫЕ РЕЗУЛЬТАТЫ

### Основные определения:
- **Группа:** множество с ассоциативной операцией, нейтральным элементом и обратными элементами
- **Подгруппа:** подмножество, замкнутое относительно операции и обратных
- **Циклическая группа:** порождаемая одним элементом
- **Смежный класс:** gH = {gh : h ∈ H}
- **Порядок элемента:** наименьшее натуральное n такое, что gⁿ = e

### Ключевые теоремы:
1. **Теорема Лагранжа:** |H| делит |G| для подгруппы H группы G
2. **Индекс подгруппы:** [G:H] = |G|/|H|
3. **Порядок элемента в ℤₙ:** ord(k) = n/gcd(k,n)
4. **Порождающие элементы в ℤₙ:** элементы, взаимно простые с n (количество: φ(n))
5. **Подгруппы циклической группы:** циклические, по одной для каждого делителя порядка

### Практические советы:
- Для проверки подгруппы: H ⊆ G и для всех a,b ∈ H выполнено ab⁻¹ ∈ H
- Для нахождения порядка элемента: найти наименьшее n такое, что gⁿ = e
- Для смежных классов: все имеют одинаковый размер |H|
- Для циклических групп: все подгруппы циклические
