## 1. Системы порождающих

### Задача 1 (55.1.а): Понятие группы
**Формулировка:** Какие из указанных числовых множеств с операциями являются группами: $\mathbb{Z}, \mathbb{Q}, \mathbb{R}, \mathbb{C}$ — одно из множеств $\mathbb{N}, \mathbb{Z}, \mathbb{Q}, \mathbb{R}, \mathbb{C}$ (с операцией сложения)?

**Решение:**
Для того чтобы множество с операцией образовывало группу, должны выполняться аксиомы замкнутости, ассоциативности, существования нейтрального элемента и существования обратного элемента.

| Множество | Замкнутость | Ассоциативность | Нейтральный элемент (0) | Обратный элемент ($-a$) | Группа? |
| :---: | :---: | :---: | :---: | :---: | :---: |
| $\mathbb{N}$ (натуральные числа) | Да | Да | Нет (0 $\notin \mathbb{N}$) | Нет | **Нет** |
| $\mathbb{Z}$ (целые числа) | Да | Да | Да ($0 \in \mathbb{Z}$) | Да ($-a \in \mathbb{Z}$) | **Да** |
| $\mathbb{Q}$ (рациональные числа) | Да | Да | Да ($0 \in \mathbb{Q}$) | Да ($-a \in \mathbb{Q}$) | **Да** |
| $\mathbb{R}$ (действительные числа) | Да | Да | Да ($0 \in \mathbb{R}$) | Да ($-a \in \mathbb{R}$) | **Да** |
| $\mathbb{C}$ (комплексные числа) | Да | Да | Да ($0 \in \mathbb{C}$) | Да ($-a \in \mathbb{C}$) | **Да** |

**Ответ:** Группами по сложению являются $\mathbb{Z}, \mathbb{Q}, \mathbb{R}, \mathbb{C}$.

### Задача 2 (55.2): Изоморфные группы
**Формулировка:** Доказать, что полугруппа $\mathbb{Z}$ с операцией $\oplus$, где $a \oplus b = a + b - 1$, является группой. Доказать, что она изоморфна аддитивной группе $\mathbb{Z}$.

**Решение:**
1.  **Доказательство, что $(\mathbb{Z}, \oplus)$ — группа:**
    *   **Нейтральный элемент:** $e = 1$, так как $a \oplus 1 = a + 1 - 1 = a$.
    *   **Обратный элемент:** $a^{-1} = 2 - a$, так как $a \oplus (2 - a) = a + (2 - a) - 1 = 1 = e$.
    *   **Ассоциативность:** $(a \oplus b) \oplus c = a + b + c - 2 = a \oplus (b \oplus c)$.
    Все аксиомы выполнены.

2.  **Доказательство изоморфизма:**
    Рассмотрим отображение $\phi: (\mathbb{Z}, \oplus) \to (\mathbb{Z}, +)$, заданное формулой $\phi(x) = x - 1$.
    *   **Биективность:** Отображение $\phi$ является биекцией, так как имеет обратное отображение $\phi^{-1}(y) = y + 1$.
    *   **Гомоморфизм:** Проверим сохранение операции:
        $$\phi(a \oplus b) = \phi(a + b - 1) = (a + b - 1) - 1 = a + b - 2$$
        $$\phi(a) + \phi(b) = (a - 1) + (b - 1) = a + b - 2$$
        Так как $\phi(a \oplus b) = \phi(a) + \phi(b)$, $\phi$ является гомоморфизмом.

**Ответ:** $(\mathbb{Z}, \oplus)$ является группой и изоморфна аддитивной группе $(\mathbb{Z}, +)$ с изоморфизмом $\phi(x) = x - 1$.

## 2. Разбиение на смежные классы

### Задача 3 (56.37.а): Смежные классы
**Формулировка:** Найти смежные классы аддитивной группы $\mathbb{Z}$ по подгруппе $n\mathbb{Z}$, где $n$ — натуральное число.

**Решение:**
Группа $G = (\mathbb{Z}, +)$, подгруппа $H = n\mathbb{Z}$. Смежный класс элемента $a \in \mathbb{Z}$ имеет вид $a + n\mathbb{Z} = \{a + nk \mid k \in \mathbb{Z}\}$.
Два элемента $a$ и $b$ принадлежат одному смежному классу тогда и только тогда, когда $a - b \in n\mathbb{Z}$, т.е. $a \equiv b \pmod{n}$.
Следовательно, смежные классы — это **классы вычетов по модулю $n$**.

**Ответ:** Смежные классы:
$$\mathbb{Z} / n\mathbb{Z} = \{0 + n\mathbb{Z}, 1 + n\mathbb{Z}, \dots, (n-1) + n\mathbb{Z}\}$$

---

### Задача 4 (56.37.в): Смежные классы
**Формулировка:** Найти смежные классы аддитивной группы $\mathbb{R}$ по подгруппе $\mathbb{Z}$.

**Решение:**
Группа $G = (\mathbb{R}, +)$, подгруппа $H = \mathbb{Z}$. Смежный класс элемента $x \in \mathbb{R}$ имеет вид $x + \mathbb{Z} = \{x + k \mid k \in \mathbb{Z}\}$.
Два действительных числа $x$ и $y$ принадлежат одному смежному классу тогда и только тогда, когда $x - y \in \mathbb{Z}$, т.е. $x$ и $y$ имеют одинаковую дробную часть.
Каждый смежный класс содержит ровно один элемент $r$ из полуинтервала $[0, 1)$, который является дробной частью всех элементов класса.

**Ответ:** Смежные классы:
$$\mathbb{R} / \mathbb{Z} = \{r + \mathbb{Z} \mid r \in [0, 1)\}$$

## 3. Гомоморфизмы

### Задача 5 (58.1.а): Гомоморфизмы
**Формулировка:** Какие из следующих отображений являются гомоморфизмами: $f: \mathbb{Z} \to \mathbb{Z}$, $f(x) = 2x$ (группы $\mathbb{Z}$ с операцией сложения)?

**Решение:**
Проверим условие гомоморфизма $f(x + y) = f(x) + f(y)$:
$$f(x + y) = 2(x + y) = 2x + 2y$$
$$f(x) + f(y) = 2x + 2y$$
Так как $f(x + y) = f(x) + f(y)$, отображение является гомоморфизмом.

**Ответ:** Отображение $f(x) = 2x$ является гомоморфизмом.

---

### Задача 6 (58.1.г): Гомоморфизмы
**Формулировка:** Какие из следующих отображений являются гомоморфизмами: $f: \mathbb{R}^* \to \mathbb{R}^*$, $f(x) = |x|$ (группы $\mathbb{R}^*$ с операцией умножения)? ($\mathbb{R}^*$ — мультипликативная группа ненулевых действительных чисел).

**Решение:**
Проверим условие гомоморфизма $f(x \cdot y) = f(x) \cdot f(y)$:
$$f(x \cdot y) = |x \cdot y|$$
$$f(x) \cdot f(y) = |x| \cdot |y|$$
Поскольку для любых действительных чисел $x, y \in \mathbb{R}^*$ выполняется свойство модуля $|x \cdot y| = |x| \cdot |y|$, отображение является гомоморфизмом.

**Ответ:** Отображение $f(x) = |x|$ является гомоморфизмом.
(Ядро гомоморфизма $\ker(f) = \{-1, 1\}$).
